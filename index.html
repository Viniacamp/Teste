O Git é diferente do GitHub; enquanto um é uma ferramenta de versionamento de código, a outra é um repositório

Comandos básicos

Depois de entrar na pasta do projeto em q deseja fazer o commit, o usuário deve dar os seguintes comandos:

- git init
- git remote add origin [link do repositório]
- git branch -M main
- git add .
- git commit -m "(descrição do commit)"
- git push -u origin main

Para clonar projetos já existentes:

- git clone [link do repositório]

Para alterar/configurar nome/email no projeto atual:

- git config user.name "Seu nome"
- git config user.email "Seu email"
